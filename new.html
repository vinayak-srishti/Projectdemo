HTML.

The HyperText Markup Language or HTML is the standerd markup language for documents designed to be displayed
in a web browser.It defines the meaning and structure of web content.It is often assited by technologies such as cascading
style sheets(css) and scripting language such as java script.

Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia
web pages.HTML describes the structure of a web page semantically and originally included cues for its appearence.

What is HTML?

HTML is a language in which most web sites are written.HTML is used to create pages and make them functional. The code
used to make them visually appealing is known as CSS and we shall focus on this later.

History of HTML.

HTML was first created by Tim Berners-Lee, Robert Cailliau, and others starting in 1989. Hyper text means that the
documents contain links that allows the reader to jump to others places in the documents altogether. A markup Language
is a way that computers speak to each other to control how text is processed and presented.To do this HTML uses two things:
tags and attributes.

What are Tags and Attributes?

Tags ans Attributes are the basics of HTML.

HTML Tags.

Tags are used to markup the start of an HTML elements and they are usually enclosed in brackets.

HTML Attributes.

Attributes contains additional pieces of information. Attributes takes the form of an opening tag and additional info
is played inside.
